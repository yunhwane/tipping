# 자동 닉네임 생성 및 기본 아바타 배정

## 배경

현재 회원가입 시 사용자가 닉네임을 직접 입력해야 합니다. 가입 절차를 간소화하기 위해 닉네임을 자동 생성하고, 기본 프로필 이미지도 자동 배정합니다.

## 요구사항

1. **닉네임 자동생성**: 회원가입 시 서버에서 한글 닉네임 자동 생성 (형용사 + 동물 + 숫자 2자리)
2. **기본 아바타 랜덤 배정**: 가입 시 8개 기본 아바타 중 1개 랜덤 배정
3. **가입 폼 간소화**: 닉네임 입력 필드 제거 (이메일 + 비밀번호만)
4. **프로필 설정에서 변경 가능**: 기존 프로필 수정 기능 유지

## 범위

### 변경
- `src/server/api/routers/auth.ts` — syncUser에서 닉네임/아바타 자동 생성
- `src/app/auth/signup/page.tsx` — 닉네임 입력 UI 제거
- `src/components/profile-settings.tsx` — 아바타 목록 8개로 조정

### 신규
- `src/lib/nickname-generator.ts` — 닉네임 생성 유틸리티

## 기술 접근

### 닉네임 생성
- 형용사(15개) + 동물(15개) + 2자리 숫자 조합
- 예: `빠른고양이42`, `용감한사자17`
- 중복 시 숫자 변경 재시도 (최대 5회), 실패 시 UUID 4자리 suffix
- 서버사이드에서 생성하여 중복 방지 보장

### 아바타 배정
- DiceBear avataaars 8개 seed 사용
- syncUser 시점에 랜덤 선택하여 image 필드에 저장

### syncUser 변경
- nickname, image 파라미터 제거
- 서버에서 자동 생성 후 DB 저장
